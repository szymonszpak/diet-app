# Diet Builder

A simple calorie and macro tracker built with **Spring Boot**.

## What can it do?

* **Finding food**: You can search for products in your own database or on the web (using Open Food Facts).
* **Speed (Hybrid Search)**: Products you’ve used before show up instantly because they are saved locally. You don't have to wait for the slow internet API every time.
* **Auto-counting**: Just enter the weight, and the app calculates calories, protein, carbs, and fats for you.
* **Daily planning**: You can create meals (like Breakfast or Lunch) and add different products to them.
* **Custom goals**: You can set your own daily calorie limit directly in the app.
